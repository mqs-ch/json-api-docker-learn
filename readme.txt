API: jsonplaceholder.typicode.com

codework:
Код обращается к api с использованием requests. Обработаны исключения в случае. если таймаут больше 10 секунд, если программа завершена пользователем или какие-то данные не получается загрузить.
JSON возвращает три категории данных:
  /users: пользователи, их id, адрес, фамилия и имя, почта, номер телефона и данные о компании;
  /posts: посты, их id, userId (автор), заголовок и тело
  /comments: комментарии (на момент загрузки и финальных тестов комментарии не загружаются, поэтому всё, что с ними связано, закомментировано)

Скрипт возвращает статистику:
  * сколько постов у пользователя
  * общие показатели
  * среднее кол-во комментариев на пост (закомменчено)
  * посты без комментариев
  * топ-5 пользователей по кол-ву постов

Обработка постов, пользователей и связи между ними идёт по свойству userId.
Обработка комментариев и постов идёт через postId (свойство комментария)
